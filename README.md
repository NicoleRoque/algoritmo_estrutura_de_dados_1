# 📚 Algoritmos e Estruturas de Dados I

Este repositório reúne os exercícios, implementações e desafios desenvolvidos durante a disciplina de Algoritmos e Estruturas de Dados I da graduação em Ciência da Computação.

O objetivo é documentar minha evolução na resolução de problemas utilizando Java, aplicando boas práticas de programação, análise de algoritmos e implementação de estruturas de dados fundamentais.

## Conteúdos abordados

* Lógica de programação
* Análise de algoritmos
* Vetores e matrizes
* Listas simplesmente encadeadas
* Listas duplamente encadeadas
* Pilhas (Stacks)
* Filas (Queues)
* Árvores genéricas
* Árvores binárias
* Percursos em árvores (pré-ordem, em ordem, pós-ordem e largura)
* Operações de inserção, remoção e busca
* Recursão
* Manipulação de ponteiros e referências
* Complexidade de algoritmos

Cada exercício foi desenvolvido com foco no entendimento dos conceitos e na prática da implementação manual das estruturas de dados, fortalecendo a base necessária para disciplinas mais avançadas e para o desenvolvimento de software.

Tecnologias utilizadas: Java • Programação Orientada a Objetos • Estruturas de Dados • Algoritmos
