# Como compilar e executar

## Compilar todos os arquivos Java

```bash
javac *.java
```

ou

```bash
javac AppListArrayOfInteger.java ListArrayOfInteger.java
```

## Executar o programa

```bash
java AppListArrayOfInteger
```

## Estrutura do projeto

```
algoritmo_estrutura_de_dados_1/
│
├── AppListArrayOfInteger.java    // Classe principal (main)
├── ListArrayOfInteger.java       // Implementação da lista
└── README.md
```

## Observações

- O arquivo que contém o `main` é `AppListArrayOfInteger.java`.
- Para executar uma classe Java, utilize apenas o nome da classe, sem a extensão `.java`.
- Caso altere uma classe, compile novamente antes de executar.
